# Photoo Prompt Gallery

**探索精选 AI 生成提示词**

一个优雅的 AI 图像生成提示词画廊，展示高质量的提示词作品

[在线预览](https://xigua222.github.io/prompt-gallery/)

</div>

---

## ✨ 功能特点

### 🎨 精美展示
- **瀑布流布局** - 响应式设计，完美适配桌面、平板、手机
- **流畅动画** - 丝滑的过渡效果和交互体验
- **高清图片** - 每张作品都经过优化，加载快速

### 🔍 强大搜索
- 按标题、提示词、作者搜索
- 按分类筛选（人像写真、场景插画、平面设计等）
- 按生成模型筛选（豆包/即梦、GPT-Image、千问、元宝等）

### 💡 提示词复制
- 一键复制完整提示词
- 查看标签和分类信息
- 支持中英文界面切换

### ❤️ 收藏功能
- 本地收藏喜欢的作品
- 快速访问收藏夹

| 分类 | 说明 |

| 人像写真 | 光感写真、社交人像、情绪人像等 |

| 手绘质感 | 手作质感、艺术风格 |

## 🛠 支持的生成模型

- **豆包 / 即梦** - 字节跳动 AI 图像生成
- **GPT-Image** - OpenAI 图像生成
- **千问** - 阿里云通义千问
- **元宝** - 腾讯混元
- **Nano Banana 2** - 其他模型

## 🚀 本地运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
